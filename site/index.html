How it Works

Theia - VSCode in the Cloud

Theia is an extensible open-source platform to develop full-fledged, multi-language IDE-like products for the cloud and the desktop using state-of-the-art web technologies.

Like VSCode

Theia not only reuses the state-of-the-art editor widget Monaco; it also provides a terminal, a file navigator, a command palette, sophisticated git integration and many more things, yielding an IDE experience like Visual Studio Code.

Cloud and Desktop IDE

Theia is entirely written in TypeScript and clearly separates the UI frontend from the backend. The same code can be run as thin-client Cloud IDE or as a classical Desktop application using Electron with either local or remote backends.

Extension System

A Theia app consists of multiple extensions. Composing a custom IDE-like product is as easy as listing the all needed extensions in a package.json file. And you can add new functionality by implementing your own extensions.

LSP and DSP

Built on the Language Server Protocol, Theia benefits from a growing ecosystem of projects to deliver intelligent editing support for many programming languages. The Debug Server Protocol brings the same for debugging.
